# Умный Медитационный Таймер
Приложение с анимацией, сохранением данных и интуитивным UI для медитаций и осознанности

## Возможности
- Запуск разных видов медитаций
- Ведение статистики
- Индивидуальные настройки


## Технологии:
- Python 3.12–3.13
- kivy

## Запуск
- `uv sync`
- `uv run app/main.py`
- или `main.exe`
